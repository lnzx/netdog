# netdog

## 监控流量
默认9.6TiB
```shell
./netdog

./netdog 5TB
```

```shell
// 使用一行可解析的格式，显示所选接口的流量摘要。输出包含 15 个字段，并使用 ;作为字段分隔符。
// 1）个字段包含输出的版本信息，如果字段结构发生变化，将在未来的vnStat版本中改变。以下字段依次为：
// 2）接口名称，
// 3）今天的时间戳，
// 4）今天的rx，
// 5）今天的tx，
// 6）今天的总数，
// 7）今天的平均流量，
// 8）本月的时间戳，
// 9）本月的rx，
// 10）本月的tx，
// 11）本月的总数，
// 12）今天的平均流量，
// 13）所有时间总rx，
// 14）所有时间总tx，
// 15）所有时间总流量。
```
